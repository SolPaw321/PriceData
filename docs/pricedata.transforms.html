pricedata.transforms.candles module

pricedata.transforms.candles.to_heikin_ashi(df: DataFrame, append: bool) → DataFrame[source]

Transform given candles to heikin ashi.

NOTE.: The value of the heikin ashi candles depends on the initial candle. The values may vary depending on the number of candles loaded.

Args:

df (pd.DataFrame): data for which candles are transformed append (bool): if true, append new columns: ha_open, ha_high, ha_low, ha_close. Otherwise, rewrite open, high,

low, close. Default is true.

Return:

Data with transformed candles.
